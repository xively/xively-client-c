Detailed Description

Xively Client's Board Support Platform (BSP) for Random Number Generator.

This file defines the Random Number Generator API needed by the Xively C Client.

The Xively C Client calls the init function before requesting any random number. The xi_bsp_rng_get function's responsibility is to provide random number each time it is called. The shutdown function will be called in case no further random numbers are required by the library. This happens at library shutdown.
